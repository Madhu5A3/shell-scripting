# uname

- `uname` Prints certain system information

- options:
	* -a => prints all the system information
	* -s => kernel name
	* -n => network node name
	* -r => kernel release
	* -v => kernel version
	* -m => machine hardware name
	* -p => processor type (non-portable)
	* -i => hardware platform
	* -o => operating system
